#Food Additives Checker

Web sederhana berbasis Flask untuk memeriksa bahan tambahan makanan berdasarkan E-number.

##Fitur
- Input E-number (misal: E100, E951)
- Menampilkan nama, fungsi, dan kategori keamanan bahan
- Tampilan web sederhana dan mudah digunakan

##Cara Menjalankan
1. Install dependensi:
pip install -r requirements.txt
2. jalankan server:
phyton app.py
3. Akses di browser: [http://127.0.0.1:5000](http://127.0.0.1:5000)
